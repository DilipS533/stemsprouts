LLE STEM — Repo

This repository contains the existing static site and a new `web/` Next.js prototype for the redesigned LLE STEM website.

Quick start (PowerShell):

```powershell
# run the prototype web app
npm run web:dev
# open http://localhost:3000
```

Folders:
- /web — Next.js + TypeScript + Tailwind prototype
- /studio — Sanity Studio scaffold (schemas only)
- existing static HTML files remain at repo root (index.html, about.html, etc.)

Next steps:
- run `npm install` in `web/` and start development server
- install Sanity and initialize the studio if you want a live CMS
