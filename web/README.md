# LLE STEM — Next.js prototype

This folder contains the Next.js prototype for the LLE STEM site. It was created as an isolated scaffold so the original HTML files remain untouched.

Quick start

1. Install dependencies:

```powershell
cd web
npm install
```

2. Run dev server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
```

Seeding CMS (Sanity)

See `web/studio/README.md` for instructions to install and seed the local Sanity studio.
# LLE STEM — Web (prototype)

This folder contains a minimal Next.js + TypeScript + Tailwind scaffold used to prototype the new LLE STEM website.

Getting started (PowerShell):

```powershell
cd web
npm install
npm run dev
# open http://localhost:3000
```

What this scaffold includes:
- Next.js (TypeScript)
- Tailwind CSS (configured)
- A starter homepage and global styles
- Design tokens file at `web/design-tokens.json`
